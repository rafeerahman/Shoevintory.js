    • Getting started

    To get started, please ensure you have the Shoevintory.js, app.css, and plot.css files in your directory.

    You can download the zip folder of the library here.

    In your HTML head, include the following

                    
                    <head>
                        <link href="https://fonts.googleapis.com/css2?family=Urbanist:ital,wght@0,100;0,200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;1,600;1,700;1,800;1,900&display=swap" rel="stylesheet">
                        <link rel="stylesheet" href="../app.css">
                        <link rel="stylesheet" href="../plot.css">

                        <script src="app/Shoevintory.js"/>
                    </head>

    In a seperate javascript file, you can initialize a new shoevintory (see instructions in API). Then you can use its various methods, and link your new javascript file within your HTML. An example can be found in the examples.html file in the ZIP shoeventory folder. See more examples in the examples page.

    API

    ShoeFactory

    Initialization

                        
                        const shoevintoryFactory = new ShoevintoryFactory();

    Properties

    name description type
    this.instances An array holding all instances of shoevintories that were created using .create(). Array []

    Arguments

    None

    Description

    To initialize a new inventory, you must first create a new ShoeFactory class by typing the above code. Once initialized, this ShoevintoryFactory will have a property called "instances", that will store newly created inventories.

    .create()

                        
                        const shoevintory = shoevintoryFactory.create();

    Arguments: None

    Return type:

    Returns a new Shoevintory object.

    Description

    Use this method to initialize and create a new shoevintory. The instance of the shoevintory object that was created will be returned. This shoevintory instance will be stored in the ShoevintoryFactory's "instances" property.

    .get(index)

                        
                        const shoevintory = shoevintoryFactory.get(0);

    Arguments:

    name description type properties
    index The index (0 to n of .create() calls) of the shoevintory you want to access. int None

    Return type:

    Returns a new Shoevintory object.

    Description

    When a shoevintory is created using .create(), that instance is stored in an array. The index of the shoevintory you call on .get() will be returned.

    Shoevintory

    Intiailization

                        
                        let shoevintory = shoevintoryFactory.create()
                        // returns a new Shoevintory object.

    Properties

    name description type properties
    this.shoevintoryFactory A reference to the shoevintoryFactory that created this shoevintory instance. ShoevintoryFactory object
    this.sneakers An array of shoe objects that are in this Shoevintory instance. Array of shoe objects, Array[Shoe object] shoe object format
    this.table The table inventory belonging to this shoevintory instance. Table object (private)
    this.plot A plot that will help create charts/graphs. Plot object (private)

    Return type

    Shoevintory object.

    Description

    Properties stored in a shoevintory instance.

    .insertTable(data, parentElement)

                        
                        
                        // Example of initializing data
                        // The "data array" and "include object" can be left as an empty array and empty object if you want, however I included some data here.
                        
                        const data = {
                            data: [
                                {
                                    img: "https://images.stockx.com/images/Air-Jordan-1-Retro-High-White-University-Blue-Black-Product.jpg?fit=fill&bg=FFFFFF&w=700&h=500&fm=webp&auto=compress&q=90&dpr=2&trim=color&updated_at=1611777406", 
                                    name: "Air Jordan 1", 
                                    brand: "Jordan", 
                                    date: "2022/04/17", 
                                    cost: 300, 
                                    salePrice: 340
                                }
                            ],
                            columnTitles: ["Image", "Sneaker name", "Brand", "Date purchased", "Cost", "Sale price"],
                            include: {
                                profits: true,
                            }
                        }
                        shoevintory.insertTable(data)

    Arguments

    name description type
    data An object holding the initial shoevintory table information. Object
    parentElement (optional) The parent element node to append the table into. If no parentElement is given, the table will be appended into the HTML Body. DOM Node

    Properties

    Argument 1 (data) format

    property name description type properties
    data (*required) An object holding the initial shoevintory shoe data. Array of shoe objects, [Object] shoe object format
    columnTitles (*required) An array of static titles in the order of each shoe property. The length of column titles array and the length of a shoe object must be equal. Array of strings, [String]
    include (*required) An object of properties to include (see include object format). Object include object format

    shoe object format

    property name description type
    name (*required) The name of the shoe. String
    date (*required) The date the shoe was acquired. String
    brand (*required) The name of the shoe's brand. String
    cost (*required) The amount that was paid for the shoe. int
    salePrice (*required) The price that the shoe was sold for. int
    any Include any other properties that you want to include in the table. Ensure that you give appropriate column titles for any seperate properties. any

    "include" object format

    property name description type
    profits Set to either true or false. boolean

    Return type

    None

    Description:

    After creating a shoevintory, you can insert a new table with the specified data format as seen above. The table DOM element can be accessed via .getTable().

    .insertRow(shoe)

                    
                    const shoe = {
                        img: " ",
                        name: "Nike Dunk Low Black/White", 
                        brand: "Nike", 
                        date: "2022/04/17", 
                        cost: 220, 
                        salePrice: 325
                    }
                    shoevintory.insertRow(shoe)

    Arguments

    name description type properties
    shoe An object matching the "shoe object" format. Object shoe object format

    Return type

    None

    Description

    Use this method to insert a new shoe into your shoevintory and the DOM.

    .deleteRow(rowId)

                    
                    shoevintory.deleteRow(1)

    Arguments

    name description type
    rowId The row number seen in the DOM's shoevintory table. int

    Return type

    None

    Description

    Removes a shoe from the shoevintory and the DOM.

    .editRow(rowId, newShoeData)

                    
                    shoevintory.editRow(1, {name: "foo"})
                    // This will edit the sneaker's name at row 1 to "foo".

    Arguments

    name description type
    rowId The row number seen in the DOM's shoevintory table. int
    newShoeData An object with any properties that match the shoe object's format. Object

    Return type

    None

    Description

    Edit a shoes properties at a specific row.

    .makeDraggable()

                    
                    shoevintory.makeDraggable()
                    // This will allow every row to become draggable.

    Arguments

    None

    Return type

    None

    Description

    Makes every row within the shoevintory draggable. This allows a user to swap rows at different places.

    .filterByBrand(brandName)

                    
                    shoevintory.filterByBrand("Adidas")

    Arguments

    name description type
    brandName The name of the brand you want to filter the table by. String

    Return type

    None

    Description

    Filters the table so that only the rows that have the brandName are shown.
